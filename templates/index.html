Note API

Method Endpoint Usage
GET /get_versions_by_id/<note_id> Get versions of note with required id
GET /get_all_notes Get all notes from database that are not deleted
GET /get_note/<note_id> Get note with provided id
POST /delete_note/<note_id> Delete note with provided id - doesn't delete note versions!
POST /update_note/<note_id> Updates note with provided id
POST /create_note Creates note with parameters from request.form
